The following fields are searchable for you:
name, Alternatives, birthDate, birthLocation, deathDate, deathLocation, Abstract, Publication (not available is the field birthYear)
To search in a specific field use the correct syntax:
fieldname:keyword
Example: birthLocation:hamburg
This query will return all entities with the word 'hamburg' in the field birthLocation.
Please note that the fieldnames have to be spelled correctly in order for your search to work.

---

AND, OR, NOT operators can be used in your query
To search with operators use the correct syntax:
keyword operator keyword
Example: stuttgart AND hamburg
This query will return all entities with the word 'hamburg' and 'stuttgart' in any field.

---

Wildcards
? can be used to replace one character
* can be used to replace zero or more characters
Examples: birthDate:10*, fel?mann
